## Build notes for Claude Code

- **Design direction:** Hybrid — one cinematic full-bleed hero (video/parallax), then clean conventional sections below (matches the MVP Trans-Logistics reference: https://mvplogistics.eu/en/main-en/).
- **Color palette suggestion:** Deep navy / steel blue + a warm amber/gold accent (common in Gulf logistics branding — signals trust + energy). Avoid generic "tech blue + white" that looks templated.
- **Typography:** A strong geometric sans for headlines (e.g. something like Inter, Sora, or Space Grotesk), paired with a clean readable body font.
- **Mobile priority:** Test the hero video/section on mobile data speeds — fall back to a static image if video feels heavy.
- **Arabic toggle:** Doesn't need to be functional for the demo — a visible AR/EN switch alone signals market awareness to a client.
- **No real company names, logos, or identifiable people** — all placeholders, as this is a portfolio demo, not a real business.
